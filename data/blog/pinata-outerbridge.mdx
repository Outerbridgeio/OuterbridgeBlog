[Pinata](https://www.pinata.cloud/) is a popular [IPFS](https://docs.ipfs.io/concepts/how-ipfs-works/) pinning service that allows users to pin file/folder/json to IPFS, a distributed system for storing and accessing files, websites, applications, and data.

Read more on Pinata [here](https://docs.pinata.cloud/faq)

Summary, you can now use Pinata on Outerbridge to:

- [x] Pin JSON
- [x] Pin File
- [x] Pin Folder
- [x] Pin via CID
- [x] List Pin By CID Jobs
- [x] Update Metadata
- [x] Unpin
